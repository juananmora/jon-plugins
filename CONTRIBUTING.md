## Contributing

¡Gracias por contribuir! Tu ayuda es clave para mantener este repositorio útil y consistente.

Este proyecto se publica bajo licencia MIT (ver [LICENSE](LICENSE)).

Ten en cuenta que este proyecto tiene un [Código de Conducta](CODE_OF_CONDUCT.md). Al participar aceptas respetarlo.

## Submitting a pull request

1. Crea una rama nueva: `git checkout -b mi-rama`
2. Realiza el cambio (y añade/actualiza tests si aplica)
3. Verifica que la validación local sigue pasando (tests/lint si existen)
4. Publica la rama y abre una pull request

Here are a few things you can do that will increase the likelihood of your pull request being accepted:

- Keep your change as focused as possible. If there are multiple changes you would like to make that are not dependent upon each other, consider submitting them as separate pull requests.
- Write a [good commit message](http://tbaggery.com/2008/04/19/a-note-about-git-commit-messages.html).
