# Seguridad

Gracias por ayudar a mantener este repositorio seguro.

## Reporte de vulnerabilidades

Si crees que has encontrado una vulnerabilidad de seguridad en este repositorio:

- No la reportes en incidencias públicas, discusiones o pull requests.
- Repórtala por un canal privado al equipo mantenedor (por ejemplo, a través del circuito de seguridad corporativo o el canal interno acordado para el proyecto).

Incluye, si es posible:

- Tipo de vulnerabilidad (p.ej. inyección, XSS, deserialización insegura)
- Rutas de ficheros afectadas
- Commit/tag/branch donde se observa
- Pasos para reproducir y configuración necesaria
- Prueba de concepto (si aplica) y análisis de impacto

El equipo mantenedor intentará confirmar y corregir el problema con el menor tiempo posible.
